단어들을 입력 받아서 각 단어의 출현 빈도를 포함한 리스트를 유지하는 프로그램을 작성한다.
단어를 입력하면 오름차순으로 Linked list에 끼워 넣는다. (단어의 순서는 strcmp로 정한다.)
같은 단어가 들어오면 출현 빈도를 update한다.
단어는 가변 길이가 되도록 포인터와 동적할당을 이용한다. (단어 입력에 사용할 버퍼는 100크기면 된>다)

Hint:

기본적으로 첫 항목으로 끼워 넣는 케이스가 있고 중간 어딘가에 끼워 넣는 케이스가 있다.
첫항목으로 끼워 넣는 경우는 시작 포인터가 0이거나, 0은 아닌데 첫 항목의 데이터보다 새
데이터가 더 작은 경우이다. (물론 같은 경우는 update만 한다.)
첫 항목으로 끼우는 경우가 아니면 p = pStart부터 시작해서 p가 가리키는 다음 항목이 있고
다음 항목이 더 작은 경우에만 p를 다음 항목으로 update해서 반복한다.
찾기가 끝나면 끼워 넣기를 하면 된다.

그림을 잘 그려보고 코딩해야한다.

main에서는
char buf[100];  //최대 단어 길이는 영문자 기준 99
while ( scanf("%s", buf) == 1 ){
    update_or_insert(&pStart, buf);
}
// 그리고는 순서대로 항목과 count를 출력한다.

예를 들어 입력된 단어가
ddd ccc aaa bbb ccc aaa bbb aaa  라면
출력은:
aaa 3
bbb 2
ccc 2
ddd 1
이 된다.
